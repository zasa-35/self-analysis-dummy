# 自己分析ワーク用 ダミーデータ集

就活向けの自己分析ワークの**教材**として使う、架空の学生のメモ集です。

- 登場人物・エピソードはすべて**創作（フィクション）**で、実在の人物・団体とは一切関係ありません。
- 個人情報は含まれていません。すべてダミーデータです。
- 後工程で「AIに強み・弱みを分析させ、ESの下書きを作る」ワークに使うことを想定しています。
- そのため、強み／弱み／エピソードの“種”があえて未整理のまま散らばっています。

## 想定する架空の学生
- 名前：佐藤リク（仮名・架空）
- 大学3年生／理系（情報系）
- 経験：コンビニバイト、塾講師バイト、フットサルサークルの運営、Web制作の長期インターン
